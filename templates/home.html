Features

Crop Recommendation

All the necessary input values are taken from the user in a form on the user interface. The values are processed and the output is recommended as a suitable crop for those data values. As of now the model could achieve an accuracy of 96%.

Disease Detection

An image is taken as an input from the user. This image is processed by the CNN technique and an output is produced that specifies the probable disease in the plant. The model can probably achieve an accuracy of 90%.

Adaptive Learning

Our machine learning model gets smarter with you! It analyzes your field data to recommend optimal crops and identify diseases. This adaptive learning ensures our recommendations stay precise as your farming practices evolve.

Model Accuracy

Random Forest Algorithm

Random Forest Algorithm takes less training time as compared to other algorithms. It predicts output with high accuracy, even for the large dataset it runs efficiently. Random Forest Algorithm can also maintain accuracy when a large proportion of data is missing.

Convolutional Neural Network

CNN leverages hierarchical pattern recognition, allowing them to automatically learn and extract features directly from raw pixel data, eliminating the need for manual feature engineering. It preserves spatial relationships within images through the use of convolutional layers, enabling them to effectively capture local patterns and structures.
